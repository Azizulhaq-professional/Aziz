# 🐍 Python Functions – Dynamic Average Calculator

This notebook demonstrates the fundamentals of **Python functions** and their real‑world applications.  
It builds from basic examples to a professional‑style program, showing how functions make code reusable, clean, and industry‑ready.

---

## 📘 Concepts Covered
- **Basic Functions** – no input, no return  
- **Functions with Parameters** – single and multiple inputs  
- **Return Values** – sending results back for reuse  
- **Default Parameters** – flexible function calls  
- **List Inputs** – processing collections of data  
- **Multiple Return Values** – returning tuples  
- **Nested Functions** – calling functions inside functions  
- **Lambda Functions** – short, inline functions used in Data Science and ML pipelines  
- **Error Handling** – professional use of `try/except`  
- **Main Function** – industry standard entry point for large projects
