# Week 02 – Deep Learning for NLP

This folder contains notebooks that build upon NLP foundations by introducing deep learning architectures. The modules progress from basic neural networks to advanced transformer models, culminating in applied systems such as question answering.

## 📂 Contents
- **07_Using_Neural_Networks_for_Text.ipynb**  
  Applying feedforward neural networks to text classification tasks.

- **08_RNN_&_LSTM.ipynb**  
  Implementing recurrent neural networks (RNNs) and long short-term memory (LSTM) models for sequential text processing.

- **09_Transformers+Attention_Mechanism+Intro_to_BERT.ipynb**  
  Exploring transformer architecture, attention mechanisms, and an introduction to BERT for contextual language understanding.

- **10_AI_Question_Answering_System.ipynb**  
  Building a question answering system leveraging transformer-based models.

## 🎯 Learning Outcomes
- Understand how deep learning enhances NLP beyond traditional methods.  
- Apply feedforward neural networks to text classification.  
- Implement RNNs and LSTMs for sequence modeling.  
- Gain hands-on experience with transformers and attention mechanisms.  
- Build a practical AI question answering system using modern architectures.

## 🚀 Next Steps
This module sets the stage for advanced chatbot development, multi-role AI assistants, and production-ready deployment covered in subsequent weeks.
