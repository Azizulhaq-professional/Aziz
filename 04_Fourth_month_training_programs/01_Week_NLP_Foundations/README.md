# Week 01 – NLP Foundations

This folder contains the foundational notebooks for Natural Language Processing (NLP) training. The modules progress from basic text handling to model improvement and semantic representation, providing a structured learning path for applied NLP.

## 📂 Contents
- **01_NLP_Fundamentals.ipynb**  
  Introduction to NLP concepts, preprocessing techniques, and text handling basics.

- **02_Text_Vectorization.ipynb**  
  Methods for converting text into numerical vectors (Bag of Words, TF‑IDF, etc.).

- **03_Spam_Detection.ipynb**  
  Applied classification model for detecting spam messages using vectorized text.

- **04_Real_Dataset_Visualization_Accuracy_Improvement.ipynb**  
  Visualization of real datasets and strategies to improve model accuracy.

- **05_Model_Improvement.ipynb**  
  Techniques for refining NLP models, including hyperparameter tuning and evaluation.

- **06_Word_Embeddings.ipynb**  
  Introduction to word embeddings (Word2Vec, GloVe) for semantic representation.

## 🎯 Learning Outcomes
- Understand the fundamentals of NLP and preprocessing workflows.  
- Apply text vectorization techniques for feature extraction.  
- Build and evaluate a spam detection model.  
- Visualize real datasets and apply accuracy improvement methods.  
- Explore strategies for model optimization.  
- Gain hands‑on experience with word embeddings for semantic analysis.

## 🚀 Next Steps
This foundation sets the stage for advanced modules such as RNNs, Transformers, and building production‑ready AI assistants in later weeks.
