# 📖 Python Dictionaries – Student Record Manager

This notebook provides **complete training on Python dictionaries**, one of the most important data structures in programming.  
It introduces dictionary basics step by step and concludes with a professional‑style mini project: **Student Record Manager**.

---

## 📘 Concepts Covered
- **What is a Dictionary?** – storing data in key‑value pairs  
- **Accessing Data** – retrieving values by keys  
- **Adding Data** – inserting new key‑value pairs  
- **Updating Data** – modifying existing values  
- **Looping Through Dictionary** – iterating over keys and values  
- **Real‑World Analogy** – labels and actual data (e.g., `name → Ali`)  

---

## 🧩 Mini Project: Student Record Manager

A professional program that demonstrates how dictionaries can be applied in real systems.

### Features
- Store student data in dictionaries  
- Add multiple students to a list  
- Display all student records neatly  
- Calculate average marks across all students
