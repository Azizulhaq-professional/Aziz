# 🌐 APIs + JSON – Student API Simulator

This notebook provides **complete training on APIs and JSON**, two fundamental concepts in modern programming.  
It introduces how APIs enable communication between systems, explains JSON as a data format, and concludes with a professional‑style mini project: **Student API Simulator**.

---

## 📘 Concepts Covered
- **What is an API?** – Application Programming Interface, enabling communication between client and server  
- **Real Examples** – Weather apps, payment systems, AI tools  
- **Client–Server Analogy** – Program = client, API = server  
- **What is JSON?** – Lightweight data format similar to Python dictionaries  
- **Example JSON Structure**:
  ```json
  {
      "name": "Ali",
      "marks": 80,
      "city": "Islamabad"
  }
- Why JSON Matters? – Used in APIs, databases, AI models, and web apps
# 🧩 Mini Project: Student API Simulator (Local JSON System)

- Add student records interactively
