# 🟢 Week 01 – Data Preparation & Feature Engineering

## 📌 Overview
This module focuses on preparing high-quality datasets, the foundation of any successful machine learning project. It introduces essential techniques for cleaning, transforming, and engineering features to improve model performance.

## 📂 Contents
- **Data Cleaning**: Handling missing values, duplicates, and inconsistencies  
- **Feature Engineering**: Creating meaningful features to capture hidden patterns  
- **Data Preparation**: Scaling, encoding, and splitting datasets for ML workflows  

## 🎯 Learning Outcomes
- Understand the importance of clean, structured data  
- Apply preprocessing techniques for real-world datasets  
- Build a strong foundation for advanced ML modeling
