# 🔵 Week 02 – Model Building & Initial Projects

## 📌 Overview
This module focuses on training machine learning models and applying them to real datasets. It introduces core techniques for model evaluation, classification, and scaling to ensure reliable performance.

## 📂 Contents
- **Model Training**: Building supervised learning models  
- **Evaluation**: Understanding overfitting vs. underfitting  
- **Classification**: Customer churn prediction case study  
- **Feature Scaling**: Standardization and normalization for consistency  

## 🎯 Learning Outcomes
- Gain practical experience in training ML models  
- Apply evaluation metrics to assess performance  
- Implement classification workflows on real-world data  
- Understand the role of scaling in improving accuracy
