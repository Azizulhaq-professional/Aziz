## 🧩 Mini Project: Sales Data Analyzer

A professional project that demonstrates how Pandas can automate sales analysis compared to manual Excel workflows.

### Features
- Create and upload a CSV file (`sales_data.csv`)  
- Load dataset into a Pandas DataFrame  
- Explore dataset with `head()` and `info()`  
- Perform basic analysis:
  - Calculate total sales per record  
  - Compute total revenue  
  - Find average product price  
  - Group sales by city
