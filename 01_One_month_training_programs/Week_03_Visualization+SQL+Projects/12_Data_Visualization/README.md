# 📊 Data Visualization – Sales Insights with Matplotlib

This notebook demonstrates how to use **Matplotlib** for data visualization, turning cleaned datasets into clear, visual insights.  
It introduces bar charts, line charts, pie charts, and histograms to present sales data effectively.

---

## 📘 Concepts Covered
- **Installing & Importing Matplotlib**  
- **Loading & Cleaning Data:** Handle missing values, remove duplicates, create `total` column  
- **Debugging Columns:** Ensuring correct column names for visualization  
- **Visualization Techniques:**  
  - Bar chart – Sales by product  
  - Line chart – Sales trend  
  - Pie chart – City contribution  
  - Histogram – Price distribution  

---

## 🧩 Mini Project: Sales Data Visualization

A professional project that demonstrates how to transform sales data into **visual insights** for business decision-making.

### Features
- Upload and clean dataset (`sales.csv`)  
- Create calculated column (`total = price * quantity`)  
- Generate multiple charts for analysis  
- Modular visualization function for reusability
