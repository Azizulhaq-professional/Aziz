# Week 01 – Python Basics

This folder contains the first week of the **One-Month Python Training Program**, focusing on foundational concepts of Python programming. Each module introduces core skills through practical exercises and mini-projects.

## 📂 Folder Structure
- **01_User Profile Generator**  
  Learn how to collect user input, process data, and generate simple outputs using Python basics.

- **02_Lists & Loops**  
  Practice working with lists, iterations, and control flow to handle repetitive tasks efficiently.

- **03_Functions**  
  Understand modular programming by defining and using functions to organize reusable code.

- **04_Dictionaries**  
  Explore key-value data structures for storing and managing structured information.

- **05_File Handling**  
  Gain experience in reading from and writing to files, enabling persistent data storage.

- **README.md**  
  Documentation summarizing the week’s modules and learning outcomes.

## 🎯 Learning Outcomes
By completing this week, you will be able to:
- Write basic Python scripts with user input/output.  
- Apply loops and conditional logic to automate tasks.  
- Create and use functions for modular code design.  
- Manage data using lists and dictionaries.  
- Perform file operations for data persistence.  

---

This week builds the foundation for advanced topics in the following weeks, preparing you for real-world problem-solving with Python.
