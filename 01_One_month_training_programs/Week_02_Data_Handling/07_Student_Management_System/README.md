# 🎓 Student Management System – JSON Based

This notebook demonstrates how to build a **menu-driven Student Management System** in Python.  
It uses **JSON files for permanent data storage**, combining Python fundamentals with real-world application design.

---

## 📘 Concepts Covered
- **JSON File Handling** – read and write structured data  
- **Functions** – modular design for each feature  
- **Lists & Dictionaries** – storing multiple student records  
- **Loops** – iterating through records  
- **Error Handling** – managing missing files and invalid input  
- **Menu-Driven Flow** – interactive system with multiple options  

---

## 🧩 Features
- Add new student records (name, marks, city)  
- View all students stored in JSON  
- Search student by name  
- Calculate average marks of all students  
- Exit the program gracefully
