# Week 02 – Data Handling

This folder contains the second week of the **One-Month Python Training Program**, focusing on data handling, APIs, and analysis techniques. Each module builds practical skills for working with structured and unstructured data.

## 📂 Folder Structure
- **06_APIs + JSON**  
  Learn how to fetch, parse, and process data from APIs using JSON format.

- **07_Student Management System**  
  Apply Python data handling concepts to design a simple management system with CRUD operations.

- **08_Pandas + Data Analysis**  
  Explore the Pandas library for efficient data manipulation, aggregation, and analysis.

- **09_Data Cleaning + Filtering**  
  Practice cleaning messy datasets, handling missing values, and filtering data for insights.

- **10_Advanced Data Analysis**  
  Perform advanced operations such as grouping, merging, and statistical analysis with Pandas.

- **README.md**  
  Documentation summarizing the week’s modules and learning outcomes.

## 🎯 Learning Outcomes
By completing this week, you will be able to:
- Work with APIs and JSON data formats.  
- Build small-scale systems for managing structured data.  
- Use Pandas for data manipulation and exploratory analysis.  
- Clean and filter datasets to prepare them for deeper insights.  
- Apply advanced analysis techniques for real-world scenarios.
