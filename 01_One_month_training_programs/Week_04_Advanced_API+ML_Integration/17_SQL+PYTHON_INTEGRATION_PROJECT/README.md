# Sales Database Analysis System (Python + SQL)

## 📌 Project Overview
This project demonstrates how to integrate **SQL with Python** for real-world sales data analysis.  
It builds a relational database, executes SQL queries from Python, and leverages **Pandas** and **Matplotlib** for analysis and visualization.

---

## 🎯 Learning Goals
- Connect Python to a SQL database.
- Store and query structured data using SQLite.
- Read SQL results into Pandas DataFrames.
- Perform business analysis and visualization.

---

## ⚙️ Technology Stack
- **Python** → scripting and analysis  
- **SQLite** → relational database  
- **Pandas** → data manipulation  
- **Matplotlib** → visualization  

---

## 🛠️ Steps Implemented
1. **Database Creation**  
   - Built `customers`, `products`, and `orders` tables with primary keys.  

2. **Data Insertion**  
   - Added sample customers (Ali, Sara), products (Laptop, Mobile), and orders.  

3. **SQL Queries via Python**  
   - Joined tables to show customer, product, quantity, and total price.  

4. **Analysis with Pandas**  
   - Calculated total revenue (₨200,000).  
   - Identified top customer (Ali).  

5. **Visualization**  
   - Bar chart of revenue by customer using Matplotlib.  

6. **Connection Management**  
   - Closed database connection after operations.  

---

**Database → SQL → Python → Pandas → Visualization**
